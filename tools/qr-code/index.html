QR코드 생성기

URL, 텍스트, Wi-Fi, 연락처, 이메일, 전화/SMS를 QR코드로 변환합니다.

  • QR 생성기
  • 사용 가이드
QR코드 내용 입력
http:// 또는 https://를 포함하면 대부분의 카메라 앱에서 바로 열립니다.
표준 포맷: WIFI:T:WPA;S:MyNetwork;P:MyPassword;;
생성된 QR코드는 입력한 내용을 그대로 담는 정적 QR입니다. 공개 장소에 부착하기 전 실제 스마트폰 카메라로 스캔해 내용과 가독성을 확인하세요.

QR코드란?

QR코드는 URL, 텍스트, 연락처 같은 정보를 사각형 패턴에 저장해 스마트폰 카메라로 빠르게 읽을 수 있는 2차원 바코드입니다.

QR코드 활용법

  • 명함
    연락처를 vCard로 공유
  • 메뉴판
    온라인 메뉴 URL 연결
  • 마케팅
    이벤트 페이지 이동
  • Wi-Fi 공유
    SSID와 비밀번호 전달
