도구 (유틸리티)

브라우저에서 바로 쓰는 무료 온라인 도구 모음

클라이언트 실행 모바일 최적화 파일 다운로드 지원
생성 도구
QR코드 생성기 ✅
URL, 텍스트, Wi-Fi, 연락처, 이메일을 QR코드로 변환. 색상 설정과 PNG 다운로드 지원.
전자서명 생성기 ✅
직접 그리기, 텍스트 서명, 도장 생성을 지원. 투명 배경 PNG와 흰 배경 JPG 다운로드.
이미지 워터마크 ✅
텍스트 또는 로고 워터마크를 이미지에 추가. 투명도, 회전, 반복 패턴과 PNG/JPG 다운로드 지원.
이미지 색상 추출기 ✅
이미지에서 HEX, RGB, HSL, CMYK 컬러 코드를 추출. 주요 색상 팔레트 자동 생성.
이미지 크기/포맷 변환기 ✅
이미지 리사이즈, SNS 프리셋, JPG/PNG/WEBP 포맷 변환과 다운로드 지원.
PDF 합치기 ✅
여러 PDF를 서버 전송 없이 브라우저에서 병합. 드래그로 순서 변경 후 다운로드.
PDF 분할 ✅
PDF에서 원하는 페이지를 추출하거나 페이지별/N페이지 단위로 나눠 다운로드.
비밀번호 생성기 준비중
길이, 문자 종류, 특수문자 포함 여부를 선택해 안전한 비밀번호 생성.
변환·계산 도구
단위환산 준비중
길이, 무게, 면적, 부피 등 자주 쓰는 단위를 빠르게 변환.
날짜계산 준비중
D-day, 날짜 차이, 영업일 계산 등 일정 관리용 날짜 도구.
