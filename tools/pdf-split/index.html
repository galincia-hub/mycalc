암호가 걸린 PDF나 손상된 PDF는 브라우저에서 열 수 없습니다. 파일은 서버로 전송되지 않고 로컬에서만 처리됩니다.

PDF 분할이란?

PDF에서 필요한 페이지만 추출하거나 여러 파일로 나누는 기능입니다. mycalc는 서버 전송 없이 브라우저에서 처리합니다.

FAQ

가능합니다. 범위 지정에 1-3, 5처럼 입력하면 해당 페이지만 새 PDF로 추출됩니다.

범위 지정 결과는 단일 PDF로, 페이지별/N페이지 분할 결과는 ZIP 파일로 다운로드합니다.

열기 암호가 있는 PDF는 브라우저에서 내용을 읽을 수 없어 분할할 수 없습니다.
