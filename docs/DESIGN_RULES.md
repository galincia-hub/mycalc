# mycalc 디자인 규칙

> tools/index.html을 레퍼런스로 전 도메인 통일 완료 (2026-05-07)

## 1단 네비 (모든 페이지 공통)
- mycalc 로고: 좌측 고정
- 메뉴: 우측 정렬 (세금 | 급여·노동 | 도구 | 부동산 | 금융·투자 | 학생 | 학부모·육아 | 생활)
- 배경: #0f172a
- 현재 도메인: active 표시
- position: sticky
- 모바일: 가로 스크롤

## 2단 네비 (서브인덱스 + 개별 페이지)
- 배경: 도메인 고유 색상의 어두운 톤
- 완성 항목: 흰색 텍스트 + 초록 점(●)
- 준비중 항목: 회색 텍스트, 점 없음
- 가로 스크롤, 좌측 정렬, sticky

## 히어로 (서브인덱스)
- 높이: 약 200px
- 배경: 도메인 고유 gradient + 미세한 도트/그리드 패턴 오버레이
- 구성: 아이콘 + 타이틀 + 설명 + 배지 pills (2~3개)

## 카드 스타일
- 흰색 배경, 0.5px 보더, border-radius: 12px
- 내부: 아이콘(40px) + 제목(굵은) + 배지 + 설명(회색)
- 배지: 완성=초록, 준비중=주황, NEW=초록
- 3열 그리드 (모바일 1열)

## 개별 계산기 페이지
- 히어로: gradient, 브레드크럼 + 배지 + 제목
- 입력 폼: 흰색, 헤더 #f0f7ff
- 결과 패널: 파란색 gradient, sticky
- 면책문구: 노란색 #fef9c3
- 폰트: Pretendard (cdn.jsdelivr.net)

## 푸터
- 면책문구 + "메인으로" 링크, 전 도메인 통일

## 과거 문제 재발 방지
- 서브인덱스에 raw bullet list 넣지 말 것 (work, tools, parent에서 깨짐 발생)
- 메인인덱스 ↔ 서브인덱스 항목은 반드시 동시 업데이트
- 인코딩: UTF-8 (BOM 없음), <meta charset="UTF-8"> 필수
