## 페이지 1: 종합소득세 계산기 (메인)

### 소개 탭
"종합소득세 계산기로 예상되는 종합소득세를 쉽고 빠르게 계산해 보세요.
국세청 기준에 따라 종합소득세를 사전에 계산해볼 수 있도록 구성되어 있습니다."

### 신고·납부 탭
- 종합소득이 있는 사람은 다음해 5월 1일~5월 31일(성실신고확인 대상 사업자는 6월 30일)까지 신고·납부
- 종합소득: 이자·배당·사업(부동산임대)·근로·연금·기타소득
- 확정신고 제외대상:
  - 근로소득만 있고 연말정산 한 경우 (2인 이상 근로소득 등 예외 있음)
  - 직전 과세기간 수입금액 7,500만원 미만 보험모집인·방문판매원
  - 퇴직소득과 연말정산대상 사업소득만 있는 경우
  - 비과세·분리과세 소득만 있는 경우
  - 연 300만원 이하 기타소득 분리과세 선택 시

### 가산세 탭 (요약 테이블)

| 종류 | 부과사유 | 가산세액 |
|---|---|---|
| 무신고(일반) | 일반무신고 | 무신고납부세액×20% |
| 무신고(복식부기) | 일반무신고 | MAX[납부세액×20%, 수입금액×0.07%] |
| 무신고(부정) | 부정무신고 | 납부세액×40% (국제거래 60%) |
| 무신고(부정+복식부기) | 부정무신고 | MAX[납부세액×40%(국제 60%), 수입금액×0.14%] |
| 납부지연 | 미납·미달납부 | 미납세액×미납기간×0.022% |

### 계산기 UI 구조 (아코디언)

**1. 종합소득금액** (5개 소득 합산)
- 사업소득: 업종코드+경비율 방식, 사업장 추가/삭제, 단순/기준경비율 선택
- 근로소득: 총급여액 입력 → 근로소득공제 자동 → 소득금액, 원천징수세액
- 연금소득: 총연금액 → 연금소득공제 자동 → 소득금액, 원천징수세액
- 금융소득: 이자(총이자액, 원천징수) + 배당(총배당액, 배당가산 대상, Gross-up, 원천징수)
- 기타소득: 총수입-필요경비=소득금액, 원천징수세액

**2. 소득공제금액**
- 인적공제: 본인(기본), 가족 동적 추가(관계선택, 70세이상/장애인/부녀자/한부모 체크)
- 보험료·주택마련: 국민연금, 공무원연금, 건강보험, 고용보험, 주택임차차입금, 장기주택저당차입금, 청약저축
- 신용카드 등: 전통시장, 대중교통, 현금영수증·직불카드, 신용카드, 도서공연
- 기타 소득공제, 종합한도 초과액(2,500만원)

**3. 세액공제금액**
- 혼인·자녀: 혼인세액공제(50만), 자녀 기본(1명25만/2명55만/3명+95만+40만), 출산입양(첫째30/둘째50/셋째70만)
- 연금계좌: 퇴직연금+연금저축, ISA 전환
- 기본: 표준세액공제(7만/12만/13만), 근로소득세액공제, 전자신고(2만)
- 배당 세액공제, 기타 세액감면

**4. 가산세·기납부세액**
- 원천징수세액(자동합산), 기납부세액, 가산세

**5. 결과**
- 총 납부세액 (큰 글씨)
- 세액 계산 테이블: 종합소득금액 → (-)소득공제 → 과세표준 → (×)세율 → 산출세액 → (-)세액공제 → 결정세액 → (+)가산세 → (-)기납부세액 → 납부(환급)할 세액
- 면책문구: "이 계산 결과는 모의계산의 결과이므로 참고용으로만 사용하시기 바랍니다."

## 페이지 2: 종합소득세란?

### 01 종합소득세 신고·납부
- 종합소득이 있는 사람은 다음해 5월 1일~5월 31일 신고·납부
- 확정신고 제외대상 목록 (페이지 1과 동일)
- 개인지방소득세도 함께 신고

### 02 장부의 비치·기장
- 모든 사업자는 장부를 비치·기록해야 함
- 간편장부대상자 기준:

| 업종 구분 | 직전 과세기간 수입금액 |
|---|---|
| 농업·임업·어업, 광업, 도소매업 등 | 3억원 미만 |
| 제조업, 숙박·음식점업, 건설업, 운수업 등 | 1억5천만원 미만 |
| 부동산임대업, 전문서비스업, 교육서비스업 등 | 7천500만원 미만 |

- 전문직사업자: 2007.1.1 이후 수입금액 무관 복식부기 의무
- 복식부기의무자: 간편장부대상자 이외 모든 사업자

### 03 소득금액 계산
- 장부 있는 경우: 총수입금액 - 필요경비
- 장부 없는 경우:
  1. 기준경비율 대상자 (①②중 작은 금액)
     - ① 수입금액 - 주요경비 - (수입금액 × 기준경비율)
     - ② {수입금액 - (수입금액 × 단순경비율)} × 배율(간편2.4/복식3.0)
  2. 단순경비율 대상자: 수입금액 - (수입금액 × 단순경비율)

### 04 신고를 하지 않는 경우의 불이익
- 세액공제·감면 불가
- 무신고가산세: MAX[납부세액×20%, 수입금액×0.07%]
- 납부불성실가산세: 미납세액 × 0.03% × 경과일수

## 페이지 3: 종합소득세 세율

### 세율 적용 방법
과세표준 × 세율 - 누진공제액
예시: 과세표준 30,000,000 × 15% - 1,260,000 = 3,240,000

### 세율표 (2023년 귀속 이후~) ← 현행

| 과세표준 | 세율 | 누진공제 |
|---|---|---|
| 14,000,000 이하 | 6% | — |
| 14,000,000 초과 50,000,000 이하 | 15% | 1,260,000 |
| 50,000,000 초과 88,000,000 이하 | 24% | 5,760,000 |
| 88,000,000 초과 150,000,000 이하 | 35% | 15,440,000 |
| 150,000,000 초과 300,000,000 이하 | 38% | 19,940,000 |
| 300,000,000 초과 500,000,000 이하 | 40% | 25,940,000 |
| 500,000,000 초과 1,000,000,000 이하 | 42% | 35,940,000 |
| 1,000,000,000 초과 | 45% | 65,940,000 |

### 과거 세율표 (SEO용 히스토리 콘텐츠)
- 2021~2022년 귀속: 1,200만 이하 6% ~ 10억 초과 45%
- 2018~2020년 귀속: 1,200만 이하 6% ~ 5억 초과 42%
- 2017년 귀속: ~5억 초과 40%
- 2014~2016년 귀속: ~1.5억 초과 38%
- 2012~2013년, 2010~2011년, 2009년, 2008년, 2005~2007년

→ 과거 세율표는 SEO 체류시간 콘텐츠로 효과적. 원본 사이트도 2005년까지 전부 나열.

## 페이지 4: 종합소득세 세액계산흐름도

### 흐름도 (원본은 이미지, 우리는 텍스트+다이어그램으로)

```
종합소득금액
  이자소득 + 배당소득 + 사업소득 + 근로소득 + 연금소득 + 기타소득
    ↓
(-) 소득공제
  기본공제(본인, 배우자, 부양가족)
  추가공제(경로우대, 장애인 등)
  연금보험료공제
  주택담보노후연금 이자비용공제
  특별소득공제(보험료, 주택자금)
  조특법(주택마련저축, 신용카드 등)
    ↓
= 과세표준
    ↓
(×) 세율 (6%~45%)
    ↓
= 산출세액
    ↓
(-) 세액공제/세액감면
  특별세액공제(보험료, 의료비, 교육비, 기부금, 표준세액공제)
  기장세액공제, 외국납부세액공제, 재해손실세액공제
  배당세액공제, 근로소득세액공제, 전자신고세액공제
  성실신고확인비용 세액공제, 중소기업특별세액감면 등
    ↓
= 결정세액
    ↓
(+) 가산세
  무(과소)신고가산세, 납부(환급)불성실가산세
  보고불성실가산세, 증빙불비가산세, 무기장가산세 등
    ↓
(-) 기납부세액
  중간예납세액, 수시부과세액, 원천징수세액 등
    ↓
= 납부할 세액 (환급세액)
```

## 페이지 5: 종합소득세 가산세 계산기

### UI 구조
- 가산세 종류 선택: 무신고 / 과소신고·초과환급신고 / 납부지연·환급불성실
- 부과사유 선택: 일반무신고 / 일반무신고(복식부기) / 부정무신고 / 부정무신고(복식부기)
- 국제거래 수반 여부 (부정무신고 선택 시)
- 법정신고기한 (날짜 입력)
- 기한후신고일자 (날짜 입력)
- 세액 입력 (원)
- 수입금액 입력 (복식부기 선택 시)
- 신고한 세액 (과소신고 선택 시)
- [계산하기] [다시하기] 버튼

### 계산 로직
- 무신고(일반): 세액 × 20%
- 무신고(복식부기): MAX[세액×20%, 수입금액×0.07%]
- 부정무신고: 세액 × 40% (국제거래 60%)
- 부정무신고(복식부기): MAX[세액×40%(국제 60%), 수입금액×0.14%]
- 기한후신고 감면: 1개월 이내 50%, 1~3개월 30%, 3~6개월 20%
- 납부지연: 미납세액 × 경과일수 × 0.022%
