# 종합소득세 계산기 — Claude Code 작업 지침

## 출처
- `incometax.calculate.co.kr/general-income-tax-calculator` 역공학
- 기술스택: WordPress + Bootstrap 5 + jQuery + 클라이언트 JS
- 참조 파일: `tax-calculator-formulas.js` (공식 전체 모듈)

## 계산 흐름 (원본 사이트 기준)

```
종합소득금액 (5개 합산)
├── 사업소득 (업종코드 + 경비율)
├── 근로소득 (총급여 - 근로소득공제)
├── 연금소득 (총연금 - 연금소득공제)
├── 금융소득 (이자 + 배당 + Gross-up)
└── 기타소득 (수입 - 필요경비)
    ↓
(-) 소득공제
├── 인적공제 (기본 150만/인 + 추가)
├── 보험료·주택마련
├── 신용카드 등
└── 기타 소득공제
    ↓
= 과세표준
    ↓
(×) 세율 (6~45%) - 누진공제
    ↓
= 산출세액
    ↓
(-) 세액공제
├── 혼인·자녀
├── 연금계좌
├── 표준(7만)/근로소득/전자신고(2만)
├── 배당 세액공제
└── 기타 감면
    ↓
= 결정세액
    ↓
(+) 가산세
(-) 기납부세액 (원천징수 등)
    ↓
= 최종 납부(환급) 세액
```

## 원본 사이트 UI 구조 요약

- 아코디언 UI로 소득·공제·세액공제 섹션 구분
- 사업소득은 업종코드 검색 → 경비율 자동 적용 (서버 API 호출)
- 인적공제는 가족 동적 추가/삭제
- 결과 URL 해시로 입력값 저장·복원 기능
- 사이드바에 관련 페이지 네비게이션

## 우리 사이트 제작 시 차이점

- WordPress 아님 → Jekyll + 정적 HTML
- 업종코드 API → JSON 데이터 파일로 대체
- Bootstrap 5 재사용 가능
- 결과 URL 해시 기능은 후순위
- SEO: Schema.org FinancialCalculator 마크업 추가
- 모바일 우선 반응형
